# Conditional statements

## if-statements 
if-statements in rust is very similar to if-statements in java or c#. 

```rust
    if (condition) {
        // logic
    } else if(condition1 && condition2)  {
         // logic
    } else {
        // logic
    }
```

## ternary operation

```rust
if condition {
        // logic
    } else {
        // logic
    };
```

## Match

``` rust
  
    match condition {
        // specify condition
        _ => {} // default condition
    }
    
```
